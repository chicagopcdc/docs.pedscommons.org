# Pediatric Cancer Data Commons (PCDC) Data Portal - Terms and Conditions

Data for the Common Good (D4CG) provides the Pediatric Cancer Data Commons (“PCDC”) Data Portal to assist researchers. The platform is designed to run queries against a database containing research data from consortia that contribute their data to the PCDC. For more information about the consortia we serve, visit [https://commons.cri.uchicago.edu/](https://commons.cri.uchicago.edu/).

While this platform is available to the public, the intent of the PCDC is that it will be used by researchers.
